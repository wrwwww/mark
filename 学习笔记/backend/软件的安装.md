#### 解压jdk

```
tar zxvf jdk-1.8.tar.gz
```

#### 配置环境变量

```
# 重写下面这行的jdk路径为上面解压jdk后的路径
export JAVA_HOME=/opt/jdk1.8.0_341
export PATH=$JAVA_HOME/bin:$PATH
export CLASSPATH=.:$JAVA_HOME/lib/dt.jar:$JAVA_HOME/lib/tools.jar
```

```
source .bash_profile
```

或者

```
echo "
export JAVA_HOME=/opt/jdk1.8.0_341
export PATH=$JAVA_HOME/bin:$PATH
export CLASSPATH=.:$JAVA_HOME/lib/dt.jar:$JAVA_HOME/lib/tools.jar
" > ~/.bash_profile
source .bash_profile
```
